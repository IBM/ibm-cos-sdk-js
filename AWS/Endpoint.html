Overview

The endpoint that a service will talk to, for example, 's3.us-south.objectstorage.softlayer.net'. If you need to override an endpoint for a service, you can set the endpoint on a service by passing the endpoint object with the endpoint option key:

var ep = new AWS.Endpoint('awsproxy.example.com');
var s3 = new AWS.S3({endpoint: ep});
s3.service.endpoint.hostname == 'awsproxy.example.com'

Note that if you do not specify a protocol, the protocol will be selected based on your current AWS.config configuration.
